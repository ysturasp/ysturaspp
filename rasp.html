Прокси-сервер (рекомендуется)

Для работы с официальным API может потребоваться VPN. Рекомендуется использовать прокси-сервер для работы из России.

Для получения API ключа:

  1. Откройте Linear
  2. Если еще не создан Workspace, создайте его
  3. Перейдите в Settings -> Security & access
  4. Перейдите к Personal API keys
  5. Нажмите New API key
  6. Скопируйте и вставьте ключ сюда

Почему именно Linear?

  • Linear - это мощный инструмент для управления задачами и проектами
  • Он позволяет нам автоматически синхронизировать задачи из вашего расписания с задачами в Linear, что делает процесс управления задачами более удобным и эффективным

    • Как начать:

    1. Нажмите на любое занятие в расписании
    2. Подключите API-ключ Linear
    3. Создавайте и управляйте задачами
